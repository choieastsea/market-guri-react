# market-guri-react
 front-side app for market-guri with React.js

## to start...
npm(9.2.0)과 node(v19.3.0)가 설치되어 있어야 합니다!
- git pull
- npm install (package.json과 같은 위치에서 수행)
- npm start
- http://localhost:3000

## core libraries
- CRA (typescript)
- react router dom
- @mui/material
- axios
- react-query
- recoil
